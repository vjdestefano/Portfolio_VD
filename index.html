Project 1 for Rutger's Coding Bootcamp. The app is called Cycler. It's a 3D rendered area that allows users to view gifs, Youtube videos, and Wikipedia facts right from their device. We used API calls to get data from the sites listed above, and we used googles speech-to-text to drive an interesting experience for the user!
