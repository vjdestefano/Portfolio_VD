Devs Helping Teachers

closeDevs Help Teachers Synopsis

Devs Helping Teachers is a full stack project utilizing a MERN stack. The idea behind the project was to reach an audience that truly needs help in everyway. This project scraped information off multiple sites, gave an overview and displays how helpful each link/site is by tracking user votes. A user can only vote once and what the user has voted on is tracked by a database.

LetterChanges(str) take the str parameter being passed and modify it using the following algorithm. It replaces every letter in the string with the letter following it in the alphabet (ie. c becomes d, z becomes a). Then capitalizes every vowel in this new string (a, e, i, o, u) and finally return this modified string.

Cycler

closeCycler Synopsis

Project 1 for Rutger's Coding Bootcamp. The app is called Cycler. It's a 3D rendered area that allows users to view gifs, Youtube videos, and Wikipedia facts right from their device. We used API calls to get data from the sites listed above, and we used googles speech-to-text to drive an interesting experience for the user!
